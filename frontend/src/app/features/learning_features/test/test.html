Beachten Sie: In diesem Versuch beschäftigen Sie sich nur mit eindimensionalen Bewegungen. Auch der theoretische Überblick beschränkt sich daher auf diese Betrachtung. Wenn Sie sich für die Weiterführung in mehreren Dimensionen interessieren, dann folgen Sie an gegebener Stelle den Hinweisen. Wenn ein Körper aus seiner Ruhelage ausgelenkt wird und durch eine Kraft F⃗ , die linear mit dem Abstand von der Ruhelage zunimmt, in Bewegung versetzt wird, dann sprechen wir von einer harmonischen Schwingung.
