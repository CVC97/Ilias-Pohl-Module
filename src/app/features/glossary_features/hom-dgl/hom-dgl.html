Homogene Differentialgleichung

Eine homogene Differentialgleichung beschreibt ein System, dass frei von Anregung schwingt/sich bewegt. Es wirken nur innere Kräfte (Trägheit, Dämpfung, Rückstellkraft,...), somit ist eine Seite der Gleichung null. Die homogene DGl des Pohlschen Rad beschreibt somit die freie Schwingun, deren Lösung von der Dämpfung abhängt.
